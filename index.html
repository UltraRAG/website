核心特性

UltraRAG 为研究者和开发者提供了一套完整的工具链，从数据处理、流程编排到最终的 UI 展示。

Pipeline

低代码编排复杂流程

原生支持串行、循环与条件分支等控制结构。开发者仅需编写 YAML 配置文件，即可在数十行代码内实现复杂的迭代式 RAG 逻辑。

# 安装依赖
pip install uv
uv sync

# 运行 Pipeline
ultrarag run examples/sayhello.yaml

# 启动 UI
ultrarag show ui --admin
